# スコープ・対象外

> **Status**: Active | 最終更新: 2026-03-14

本ドキュメントは、konbuのスコープ（範囲）を明確にする。

---

## スコープ定義

### 対象範囲

| カテゴリ | 対象 | 説明 |
|----------|------|------|
| 機能 | メモ | Markdown/テーブル型、タグ、横断検索 |
| 機能 | ToDo | ステータス管理、期限、タグ |
| 機能 | カレンダー | 月表示、終日/時間指定、繰り返し、iCalインポート |
| 機能 | ツールランチャー | ブックマーク管理、カテゴリ、favicon、ヘルスチェック |
| 機能 | 横断検索 | pg_trgmによる全文検索 |
| 機能 | CLI | 全リソースCRUD、JSON出力、短縮ID |
| 機能 | Export/Import | JSON/Markdown ZIPエクスポート、iCalインポート |
| ユーザー | 個人（クラウド / セルフホスト） | メール+パスワード認証、マルチユーザー対応 |
| プラットフォーム | Webブラウザ + CLI | レスポンシブWeb UI + Go CLIバイナリ |
| 提供形態 | セルフホスト（OSS） | MIT ライセンスで全機能無料公開 |
| 提供形態 | クラウド（SaaS） | 無料で全機能利用可、Sponsors 特典あり |

### 対象外

| カテゴリ | 対象外 | 理由 |
|----------|--------|------|
| 機能 | チャット・メッセージング | グループウェア機能は個人ワークスペースに不要 |
| 機能 | ワークフロー・承認 | エンタープライズ向け機能 |
| 機能 | ファイルストレージ | 専用のストレージサービスに任せる |
| 機能 | リアルタイム同期 | WebSocketは使わない設計判断 |
| ユーザー | モバイルネイティブアプリ | レスポンシブWebで対応 |

---

## 前提条件

### セルフホスト版

| 前提 | 説明 |
|------|------|
| セルフホスト環境 | Docker or ネイティブでサーバーを運用できる環境 |
| PostgreSQL 16+ | pg_trgm拡張が利用可能なPostgreSQL |
| リバースプロキシ | 本番環境ではTraefik/Caddy/nginx等でTLS終端 |

### クラウド版

| 前提 | 説明 |
|------|------|
| ブラウザ | モダンブラウザ（Chrome, Firefox, Safari, Edge） |

---

## 制約事項

| 制約 | 種別 | 説明 |
|------|------|------|
| REST API only | 技術 | gRPC, GraphQL, WebSocketは使用しない |
| sqlcによるコード生成 | 技術 | ORMは使わず、手書きSQLをsqlcで型安全に変換 |
| 個人利用が前提 | 設計 | マルチユーザー対応だがチーム機能は持たない |
