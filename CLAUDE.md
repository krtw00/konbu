# konbu

## Plane (Issue 管理)

- Plane project: `codenica/KONBU` — https://plane.codenica.dev/codenica/projects/1bb62bd0-c25d-40dc-a518-dbdfb535eb85/issues/
- Issue 起票は Plane で `KONBU-NN` として行う (GitHub Issues / Forgejo Issues は使わない)
- commit msg は `KONBU-12 fix login error` のように Issue ID を先頭に書いて連携する
- 状況確認は Web UI、 または Claude code セッション内の plane MCP tool (`mcp__plane__list_work_items` 等)
